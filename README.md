# 🎣 NeFish — Океанская рыбалка

Современный браузерный симулятор рыбалки на океане. Дневная мультяшно-реалистичная сцена, живой океан с рыбами 6 редкостей, физическая леска, магазин, улучшения и стеклянный UI.

## Стек

- **Next.js 15** (App Router) + **TypeScript**
- **Tailwind CSS** — стили и glassmorphism UI
- **Framer Motion** — анимации интерфейса
- **Zustand** — состояние игры (с сохранением в localStorage)
- **Canvas API** — рендер сцены, рыб, лески и эффектов в одном rAF-цикле

## Запуск

```bash
npm install
npm run dev      # http://localhost:3000
```

Сборка прод-версии:

```bash
npm run build && npm start
```

## Как играть

- Удочка закреплена на причале справа сверху, крючок следует за курсором.
- Леска физически (verlet-симуляция) изгибается и тянется за мышью.
- Наведите крючок на рыбу и **нажмите**, чтобы подсечь. Шанс зависит от удачи.
- Подсечённая рыба вытягивается к причалу → монеты + опыт.
- Открывайте 🛒 **Магазин** (удочки, катушки, крючки, наживки, леска) и
  ⬆️ **Улучшения** (скорость вытягивания, удача, шанс редкой рыбы, глубина, инвентарь).

## Архитектура

```
app/                 # layout, page, globals
components/
  FishingCanvas.tsx  # игровой движок: окружение, рыбы, физика лески, эффекты
  GameRoot.tsx       # композиция canvas + UI
  ui/                # Hud, Modal, ShopModal, UpgradesModal, CatchToast
data/                # fish, shop, upgrades (контент игры)
store/gameStore.ts   # Zustand store + расчёт характеристик
```

Контент (виды рыб, товары, улучшения) вынесен в `data/` — добавление новой рыбы
сводится к одной записи в `FISH_SPECIES`.
